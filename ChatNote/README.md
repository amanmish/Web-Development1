# ChatNoteApp
A Beautifully designed CRUD Blog application made with Node.js, Express , MongoDB , HTML , CSS , Bootstrap
# Features
- Show all Posts
- Read Post
- Create new post
- Edit Post
- Update Post
- Save Post
- Delete Post
# Requirements
- Node.js
- Express
- MongoDB
- Mongoose
- EJS
- body-parser
- method-override
# Run
$node app.js
